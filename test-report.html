Findings & recommendations

  • ✓ 7 / 8 archetypes pass. All AMOLED sizes (360 / 390 / 416 / 454) and all MIP round sizes (240 / 260 / 280) render the red spiral correctly, auto-scaled, with the spiral actively rotating clockwise and the white hands on top.
  • ✗ Instinct 2 fails. The 2-color MIP display can't show red — it maps to the black background, so the spiral vanishes (only 26 px changed between frames vs 26k–86k elsewhere). Fix: detect the Instinct/low-color device class and force a white (foreground-color) spiral, ideally a thinner single-width variant without the taper. The report's suggestion of a separate Instinct SKU also applies.
  • ⚠ MIP animation is a real-hardware caveat. In the simulator the spiral animates on MIP (fr955 / fēnix 7S / 7X) just like AMOLED, but real MIP hardware throttles watch-face redraws. Recommend gating the animation loop to AMOLED and falling back to a static spiral on MIP before shipping to those devices. Physical-device validation still needed for ≥1 MIP unit.
  • ℹ Build warnings. All non-fenix8 builds emit a single cosmetic warning — the 65×65 launcher icon is scaled to each device's icon size. Harmless; supply per-device icons later if desired.
